# Github user activity
Simple command line interface (CLI) to fetch the recent activity of a GitHub user and display it in the terminal. As part of the CLI the user is able to run the command line, accept the GitHub username as an argument, fetch the userâ€™s recent activity using the GitHub API, and display it in the terminal.

* Provide the GitHub username as an argument when running the CLI.

# Tech being used
* [Github API](https://docs.github.com/en/rest/activity/events?apiVersion=2022-11-28)
* [Commander](https://www.npmjs.com/package/commander#installation)
* [Node version 20.2.0](https://nodejs.org/en)
